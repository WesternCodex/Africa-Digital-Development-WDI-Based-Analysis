# Africa-Digital-Development-WDI-Based-Analysis (2000вЂ“2024)

## Project Overview

This project analyzes digital development across 54 African countries using the World Bank Development Indicators (WDI) dataset.

The analysis explores Internet adoption, mobile subscriptions, GDP per capita, and GDP growth between 2000 and 2024 to identify trends, relationships, and opportunities for digital transformation across Africa.

An interactive Power BI dashboard was developed to communicate key insights and support data-driven decision-making.

---

## Project Objectives

- Analyze digital development trends across Africa.
- Measure Internet usage and mobile connectivity.
- Examine the relationship between Internet usage and GDP per capita.
- Identify countries leading in digital adoption.
- Generate actionable insights and recommendations for policymakers and stakeholders.

## Key Findings

- The analysis covers **54 African countries** between **2000 and 2024**.

- Average Internet usage reached **44.89%** of the population.

- Average mobile subscriptions increased to **107.25 subscriptions per 100 people**, indicating that many individuals own multiple SIM cards or mobile lines.

- Average GDP per capita across the selected countries is approximately **$2.31K**.

- Internet usage has a **strong positive correlation (0.72)** with GDP per capita, suggesting that countries with higher Internet adoption generally experience higher economic output per person.

- Mobile subscriptions and Internet usage show a **moderate positive correlation (0.50)**, indicating that increased mobile connectivity supports Internet adoption but is not the only determining factor.

- Morocco recorded the **highest Internet usage (91%)** among the selected countries.

- CГґte d'Ivoire recorded the **highest mobile subscription rate (184 subscriptions per 100 people)**.

- Seychelles recorded the **highest GDP per capita (approximately $13.5K)** among the countries analyzed.

- Overall, Internet adoption, mobile connectivity, and economic performance have shown steady growth across Africa over the study period, although significant differences remain between countries.
